Hi, I'm Mike.

I'm a programmer, fixer upper and general IT guy.

I work for the National Museum of Ireland in Collins Barracks. I am a member of the ICT Department and am mainly responsible for the network and server infrastructure.

I have developed some in house applications to make it easier for staff to share information and get their work done. One of the projects was to create a display to inform visitors of upcoming events in the Museum. This was completed using PHP for the backend and HTML, CSS and Javascript to showcase the events.

Before this I worked with BlueCarrot as a PHP developer. We worked on a very large web based system for the GAA.

Skills

I have used PHP and Ruby to make intranet and web facing sites. Along with these, I used HTML, CSS, Javascript, bash, perl, sed and awk. I have used cakePHP quite a lot and also dabbled with Sinatra for some projects.

I use git for all my version control needs now but I did play around with SVN and Mercurial in the past.

I have used a number of SQL variants including MySQL, PostgreSQL and MSSQL but I haven't had to dig too deep into their performance.

I am comfortable working in Linux, Windows and MacOSX environments having supported all of them on desktop and server platforms.

I also have experience administering Cisco networking devices from switches to routers to PIX firewalls.
