Hi, I'm Mike and I'm a programmer, fixer and general IT guy.

I'm currently working with the National Museum of Ireland in Collins Barracks. I help to look after the IT infrastructure for four public sites and a number of other offices.

I have developed some in house applications to make it easier for staff to share information and get their work done. One of the projects was to create a display to inform visitors of upcoming events in the Museum. This was completed using PHP for the backend and HTML, CSS and Javascript to showcase the events.

Before this I worked with a company called BlueCarrot which did PHP development and hosting. We worked on a very large web based system for the GAA.

Skills
